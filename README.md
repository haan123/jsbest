## Javascript Benchmark Test (JSBest)
Client side testing for javascript performance, compare with multiple sample to get the best appoach.

### Dependencies
- Code Mirror
- Hogan.js
- Lodash
- Prismjs
- Benchmark
- Foundation

### Technologies & Methods
- ES6
- Sass
- Webpack
- Handler (DOM event handling)
